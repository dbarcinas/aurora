# Aurora

SpaceX data in the comfort of your Terminal.

![Demo](.github/demo.gif)

## Usage

Clone the repository then run the following command

```sh
cargo run
```
    
## Build

Build and distribute this by running the following command
```sh
cargo build --release
```
